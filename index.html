⚙️ การตั้งค่าปัจจุบัน

ค่าเหล่านี้ฝังอยู่ในไฟล์ index.html โดยตรง — ใช้งานได้ทุกเครื่อง ไม่ต้องกรอกใหม่

📝 วิธีเปลี่ยนค่า

เปิดไฟล์ index.html → หาบรรทัด CONFIG ด้านบนสุดของ <script> แล้วแก้ค่าในส่วน:

const GAS_URL = 'ใส่ URL ใหม่ที่นี่';

Gemini API Key ไม่ต้องใส่ในไฟล์นี้ — เก็บไว้ใน GAS Script Properties อยู่แล้ว ปลอดภัย ✅

จากนั้นอัพไฟล์ขึ้น GitHub ใหม่ — ทุกเครื่องที่เปิดลิงก์จะได้ค่าใหม่ทันที

⚠️ ถ้า GitHub repo เป็น Public คนอื่นจะเห็น GAS URL ได้ แต่ไม่อันตรายเท่า Key เพราะต้องผ่านสิทธิ์ Google อยู่ดี — แนะนำ repo Private เพื่อความปลอดภัยสูงสุด
